# JSON 檢查、排版與壓縮工具

一個使用 HTML、CSS、JavaScript 製作的前端小工具。

使用者可以貼上 JSON 內容，檢查格式是否正確，並進行：
- 縮排排版（Pretty Print）
- 壓縮成單行（Minify）
- 語法上色預覽（Key / String / Number / Boolean / Null）
- 一鍵複製結果
- 上傳 .json 檔案
- 下載排版後 JSON
- 即時驗證（輸入時自動檢查）

## 功能特色

- JSON 格式驗證
  - 快速判斷貼上的內容是否為有效 JSON
  - 若格式錯誤，會顯示錯誤訊息

- JSON 縮排排版
  - 將合法 JSON 轉為易讀格式（2 空白縮排）

- JSON 壓縮成單行
  - 將合法 JSON 轉為單行字串，方便傳輸或儲存

- 語法上色預覽
  - 驗證成功後會顯示語法上色
  - 可辨識 key、字串、數字、布林值、null

- 複製結果
  - 將目前文字區內容一鍵複製到剪貼簿

- 上傳 .json 檔案
  - 載入本機 JSON 檔並自動驗證
  - 驗證成功會自動套用縮排格式

- 下載排版 JSON
  - 下載目前內容的排版版本（formatted.json）

- 即時驗證
  - 使用者輸入時自動驗證 JSON 格式
  - 即時更新語法上色預覽與狀態訊息

- 清空內容
  - 一鍵清空輸入與預覽狀態

## 使用方式

1. 直接在瀏覽器開啟 index.html。
2. 將 JSON 貼到輸入區。
3. 使用下列按鈕：
    - 驗證 JSON：檢查格式是否正確
    - 格式化縮排：轉為易讀格式
    - 壓縮成單行：轉為一行 JSON
    - 複製結果：複製目前文字區內容
    - 上傳 .json：從本機載入 JSON
    - 下載排版 JSON：下載排版後內容
    - 清空：清除內容
4. 在下方語法上色預覽區查看結果。
5. 輸入中會自動進行即時驗證。

## 專案結構

- index.html：頁面結構與操作區塊
- styles.css：藍色系主題樣式與語法上色樣式
- script.js：JSON 驗證、格式化、壓縮與預覽邏輯

## 注意事項

- 本工具僅處理標準 JSON（符合 JSON.parse 規範）。
- 若貼上的內容包含註解、單引號或尾逗號，會被視為格式錯誤。
